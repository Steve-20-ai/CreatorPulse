# CreatorPulse

CreatorPulse 是一个面向影石全球达人营销场景的可解释决策原型。它把“产品能力—户外场景—本地受众—内容表达—转化结果”连接成闭环，帮助运营团队从候选达人中找到更可能引爆单品的人，并清楚回答“为什么推荐、证据是什么、还缺哪些数据”。

> 当前仓库为比赛 MVP。所有达人与投放数据均为合成演示数据，不代表任何真实个人、账号或业务结果。

## 核心差异

1. **场景图谱匹配**：不以粉丝量为中心，匹配产品能力、真实户外任务与目标受众。
2. **证据链推荐**：同时输出推荐分、关键证据、置信度与数据缺口。
3. **归因学习闭环**：用投放结果更新下一轮选人和预算建议，而不是只做事后报表。
4. **国内外统一决策**：同一套可解释框架覆盖中国与海外市场，同时按市场切换候选池、平台、语言和合规提示。

当前 MVP 覆盖中国、德国、美国、日本、英国、法国、澳大利亚、韩国等演示市场；中国市场默认执行“本土优先”，只让主要受众市场为中国的达人进入排序。中国合成样本共 12 条，覆盖抖音、B站和小红书，各平台 4 条。全部达人记录仍为合成数据。

## 黄金演示流程

```text
选择产品 / 市场 / 场景 / 目标
  → 生成达人排行榜
  → 查看证据链与数据缺口
  → 生成本地化合作内容
  → 查看归因预测与下一轮建议
```

## 快速运行

环境要求：Node.js 18 或更高版本，无第三方依赖。

```bash
npm start
```

浏览器打开：<http://localhost:4173/web/>

运行匹配算法测试：

```bash
npm test
```

```text
└─ web/                     # 交互式决策驾驶舱
```

## MVP 边界

- 不抓取或绕过任何社交平台访问限制。
- 不声称合成数据是实际达人数据或真实投放结果。
- 内容生成采用可解释模板，生产环境可替换为经审核的模型服务。
- 正式归因需接入授权 API、影石历史数据、专属链接/优惠码及对照实验。

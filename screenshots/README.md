# Screenshots

截图应来自实际运行的原型，不使用设计稿或伪造界面图。

## 生成方式

1. 在仓库根目录运行 `npm start`。
2. 打开 <http://localhost:4173/web/>。
3. 建议使用 1440 × 1000 或更大的浏览器视口。
4. 至少保存以下三张图：
   - `01-ranking-and-evidence.png`：默认排名与证据链。
   - `02-localized-content.png`：切换到“本地化内容”。
   - `03-attribution-projection.png`：切换到“归因预测”。

本次自动化环境的浏览器安全策略阻止了本地页面重新加载，因此仓库暂不包含未经验证的截图。运行者应按以上步骤从真实页面生成。
